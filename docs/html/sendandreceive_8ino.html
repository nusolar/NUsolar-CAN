void setup ( )

The Arduino Setup() routine begins the SPI protocols to output received data to the COM window, Sets the applicable CAN filters, and then calls the CAN_IO::Setup() routine. In this case, the CAN controller is set up to listen only for BMS State of Charge and motor controller velocity data packets.
