14 /*
15  * \brief Struct containing the filter info for the rx buffers.
16  *
17  * The user can specify 1 mask and 2 filters for RXB0, and 1 mask and
18  * 5 filters for RXB1.
19  *
20  * Any message which matches any of the filters on all of the bits where their respective mask contains
21  * a 1 will be accepted. e.g. If
22  * RXM0 = 010b
23  * RXF0 = 000b
24  * Then messages with IDs matching x0xb will be accepted, where x is either a 0 or a 1.
25  */
